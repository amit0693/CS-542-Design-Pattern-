The 'Test.java' class is in charge of executing the main method, through which photo albums and photo 
albums with sub menus can be added.

The program shows all the menus and all their photos at startup.

Subsequently, the user is allowed to search for a photo or exit the program, in case the user decides to 
search for a photo,
You will be prompted for a search filter (search by year, or search by keyword).

After entering the search keyword or year, all resulting photos will be displayed regardless of their 
containing album.
